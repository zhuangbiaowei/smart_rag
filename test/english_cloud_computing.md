# Cloud Computing Fundamentals

Cloud computing delivers compute, storage, and networking over the internet. It allows teams to provision resources on demand without owning physical hardware.

## Key characteristics

- On-demand self-service
- Broad network access
- Resource pooling
- Rapid elasticity
- Measured service

## Service models

- **IaaS**: Virtual machines, storage, and networking
- **PaaS**: Managed platforms for building and deploying apps
- **SaaS**: Fully managed software delivered via the web

## Deployment models

- **Public cloud**: Shared infrastructure operated by a provider
- **Private cloud**: Dedicated infrastructure for a single organization
- **Hybrid cloud**: Integration of public and private resources

## Common use cases

- Scalable web applications
- Data analytics pipelines
- Backup and disaster recovery
- Dev/test environments
