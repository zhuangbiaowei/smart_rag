# 财务分析

财务分析是通过研究和解读财务报表来评估企业财务状况和经营绩效的过程。

## 财务报表

### 资产负债表

资产负债表展示企业在特定时点的财务状况：

$$资产 = 负债 + 所有者权益$$

**主要项目**:
- **流动资产**: 现金、应收账款、存货
- **非流动资产**: 固定资产、无形资产
- **流动负债**: 应付账款、短期借款
- **非流动负债**: 长期借款、债券
- **所有者权益**: 实收资本、留存收益

### 利润表

利润表展示企业在一定时期内的经营成果：

$$收入 - 费用 = 利润$$

**主要项目**:
- **营业收入**: 主要经营活动收入
- **营业成本**: 销售产品或提供服务的成本
- **毛利**: 营业收入 - 营业成本
- **营业费用**: 销售、管理、研发费用
- **营业利润**: 毛利 - 营业费用
- **净利润**: 最终利润

### 现金流量表

现金流量表展示现金的流入和流出：

**三部分**:
- **经营活动现金流量**: 日常业务产生的现金
- **投资活动现金流量**: 购买/出售资产的现金
- **筹资活动现金流量**: 融资和还款的现金

## 财务比率分析

### 流动性比率

衡量企业短期偿债能力：

**流动比率**:
$$流动比率 = \frac{流动资产}{流动负债}$$

理想值: 1.5-2.0

**速动比率**:
$$速动比率 = \frac{流动资产 - 存货}{流动负债}$$

理想值: 1.0 或以上

### 营运能力比率

衡量资产利用效率：

**存货周转率**:
$$存货周转率 = \frac{营业成本}{平均存货}$$

**应收账款周转率**:
$$应收账款周转率 = \frac{营业收入}{平均应收账款}$$

**总资产周转率**:
$$总资产周转率 = \frac{营业收入}{平均总资产}$$

### 盈利能力比率

衡量盈利能力：

**毛利率**:
$$毛利率 = \frac{营业收入 - 营业成本}{营业收入} \times 100\%$$

**净利率**:
$$净利率 = \frac{净利润}{营业收入} \times 100\%$$

**资产回报率 (ROA)**:
$$ROA = \frac{净利润}{平均总资产} \times 100\%$$

**净资产收益率 (ROE)**:
$$ROE = \frac{净利润}{平均所有者权益} \times 100\%$$

### 偿债能力比率

衡量长期偿债能力：

**负债比率**:
$$负债比率 = \frac{总负债}{总资产} \times 100\%$$

**权益乘数**:
$$权益乘数 = \frac{总资产}{所有者权益}$$

**利息保障倍数**:
$$利息保障倍数 = \frac{息税前利润}{利息费用}$$

## 杜邦分析法

杜邦分析法将 ROE 分解为三个部分：

$$ROE = \text{净利率} \times \text{总资产周转率} \times \text{权益乘数}$$

**组成部分**:
1. **盈利能力**: 净利率
2. **营运能力**: 总资产周转率
3. **财务杠杆**: 权益乘数

## 趋势分析

比较不同时期的财务数据，识别趋势：

- 水平分析：比较相邻时期的变化
- 垂直分析：将各项占基准项的百分比
- 共同比分析：利润表各项目占营业收入的百分比
- 共同规模分析：资产负债表各项目占总资产的百分比

## 同业比较分析

与同行业企业进行比较：

- 比率比较
- 市场份额
- 规模比较
- 成长性比较

## 财务预测

### 销售预测

- 历史趋势分析
- 市场调查
- 行业分析
- 经济因素

### 盈利预测

基于销售预测和成本结构：

$$预测利润 = 预测收入 - 预测成本$$

### 现金流预测

预测未来的现金流入和流出：

- 经营活动现金流
- 投资活动现金流
- 筹资活动现金流

## 估值方法

### 相对估值法

- 市盈率 (P/E): $\frac{股价}{每股收益}$
- 市净率 (P/B): $\frac{股价}{每股净资产}$
- 市销率 (P/S): $\frac{股价}{每股营业收入}$

### 绝对估值法

- **折现现金流模型 (DCF)**:
  $$企业价值 = \sum \frac{自由现金流}{(1 + 折现率)^n}$$

- **股息折现模型 (DDM)**:
  $$股票价值 = \sum \frac{股息}{(1 + 折现率)^n}$$
