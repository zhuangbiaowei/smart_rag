# 微积分基础

微积分是数学的一个分支，主要研究变化的累积率和量的累积。它由微分和积分两个主要部分组成。

## 微分学

微分学研究函数的变化率。

### 导数

导数表示函数在某一点的变化率。对于函数 $f(x)$，其导数定义为：

$$f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$$

### 基本导数公式

- 常数规则：$\frac{d}{dx}(c) = 0$
- 幂规则：$\frac{d}{dx}(x^n) = nx^{n-1}$
- 指数规则：$\frac{d}{dx}(e^x) = e^x$
- 对数规则：$\frac{d}{dx}(\ln x) = \frac{1}{x}$

### 导数的应用

- **速度**: 位移对时间的导数
- **加速度**: 速度对时间的导数
- **优化**: 找到函数的最大值和最小值
- **斜率**: 切线的斜率

## 积分学

积分学研究量的累积。

### 不定积分

不定积分是导数的逆运算：

$$\int f(x) dx = F(x) + C$$

其中 $F(x)$ 是 $f(x)$ 的原函数，$C$ 是积分常数。

### 定积分

定积分计算函数在区间上的累积量：

$$\int_{a}^{b} f(x) dx = F(b) - F(a)$$

### 基本积分公式

- $\int 1 dx = x + C$
- $\int x^n dx = \frac{x^{n+1}}{n+1} + C$ (n ≠ -1)
- $\int e^x dx = e^x + C$
- $\int \frac{1}{x} dx = \ln|x| + C$

## 微积分基本定理

微积分基本定理连接了微分和积分：

$$\frac{d}{dx} \int_{a}^{x} f(t) dt = f(x)$$

这意味着积分和微分是互逆的运算。

## 应用领域

### 物理学

- **运动学**: 计算位移、速度和加速度
- **功和能**: 功是力对位移的积分
- **电磁学**: 计算电场和磁场

### 经济学

- **边际分析**: 边际成本和边际收益
- **弹性**: 价格弹性计算

### 工程学

- **结构分析**: 计算梁的弯曲
- **流体动力学**: 计算流量和压力

### 统计学

- **概率密度函数**: 计算概率
- **期望值**: 积分计算期望

## 技巧

### 积分技巧

1. **换元积分法**: 通过变量替换简化积分
2. **分部积分法**: 适用于乘积形式的被积函数
   $$\int u dv = uv - \int v du$$
3. **部分分式分解**: 将有理函数分解为简单分式

微分方程是微积分的重要应用，用于描述各种物理和工程现象。
