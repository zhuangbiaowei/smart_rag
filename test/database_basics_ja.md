# データベース入門

データベースは情報を整理して保存し、必要なときに効率よく検索できる仕組みです。業務システムやWebサービスの基盤として広く利用されています。

## 主な種類

- **リレーショナルデータベース**: 表形式で管理し、SQLで操作する
- **ドキュメントデータベース**: JSONのような構造で柔軟に保存する
- **キー・バリューストア**: 高速な読み書きに特化する

## 基本用語

### テーブル

行と列で構成され、行はレコード、列は属性を表します。

### 主キー

各レコードを一意に識別するための値です。

### インデックス

検索速度を向上させるための構造です。適切に設計することで性能が改善します。

## SQLの基本

```sql
SELECT name, price
FROM products
WHERE price > 1000
ORDER BY price DESC;
```

## トランザクション

複数の操作を一つのまとまりとして扱い、整合性を保ちます。失敗時はロールバックされます。

## 運用のポイント

- バックアップと復元手順の整備
- 権限管理と監査ログ
- パフォーマンスの監視とチューニング

データベースの基礎を理解することで、安定したシステム設計につながります。
